CustomFirmata
=============
This is an Arduino sketch.

Added "temperature &amp; humidity reading" and "IR emission" features to the StandardFirmata sketch.
Both uses SysEx `STRING_DATA` command.

## Requirements
- "Grove Temperature & Humidity Sensor Pro (SEN51035P)" is connected to `A0` pin
- "Grove - Infrared Emitter" (WLS12148P) is connected to `Digital 12` pin

If you want to change pin mapping, please fork and modify :)
